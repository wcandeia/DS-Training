Estatística: A Ciência das Escolhas¶

Introdução¶

Em um teste Stroop, participantes são apresentados a uma lista de palavras, e cada uma das palavras está de uma cor. A tarefa dos participantes é dizer em voz alta a cor em que a palavra foi exibida. A tarefa tem duas condições: uma condição de palavras congruentes e uma condição de palavras incongruentes. Na condição de palavras congruentes, a palavra exibida são nomes de cores cujos nomes são a mesma cor em que a palavra foi exibida: por exemplo, RED, BLUE. Na condição de palavras incongruentes, as palavras apresentadas são nomes de cores cujos nomes não são as mesmas cores em que as palavras foram apresentadas: por exemplo, <font color=purple>PURPLE</font>, <font color=orange>ORANGE</font>. Em cada caso, medimos o tempo que demora para o indivíduo falar os nomes em uma lista das cores de tamanhos iguais. Cada participante realizará o teste e tem o tempo medido para a realização do teste nas duas condições.

Questões para Investigação¶

Questão 1: Qual é a nossa variável independente? Qual é a nossa variável dependente?

Resposta: A variável independente é o conjunto de palavras em condição congruente ou incongruente. A variável dependente é o tempo que demora para o indivíduo falar os nomes em uma lista das cores de tamanhos iguais.

Questão 2a: Qual seria um conjunto apropriado de hipóteses para essa tarefa? Que tipo de teste estatístico você espera executar? Justifique suas escolhas.

Resposta: Existem várias possibilidades: O tempo que demora para o indivíduo falar os nomes em uma lista das cores de tamanhos iguais muda significativamente nas duas condições? O tempo é maior ou menor em uma das condições? Palavras conhecidas pelo participante afetam o desempenho da leitura? O desempenho é significativamente diferente em relação ao gênero do participante?

Exemplo com notação matemática: Hipótese nula: $H_0$: O tempo que demora para o indivíduo falar os nomes em uma lista das cores de tamanhos iguais não muda significativamente nas duas condições. ($\mu_c$ = $\mu_{inc}$, onde: $\mu_c$ é a média de tempo para condição congruente e $\mu_{inc}$ a média de tempo para a condição incongruente.)

Hipótese alternativa: $H_A$: O tempo que demora para o indivíduo falar os nomes em uma lista das cores de tamanhos iguais é maior na condição incongruente. ($\mu_c$ < $\mu_{inc}$, onde: $\mu_c$ é a média de tempo para condição congruente e $\mu_{inc}$ a média de tempo para a condição incongruente.)

A variância da população é desconhecida e é necessário estima-la a partir da amostra. Também temos que o tamanho da amostra é pequeno (n < 30). Assim, o teste T de Student pode ser mais apropriado.

Questão 2b: Resposta: O tipo das amostras é pareado. Utilizamos teste t pareado (amostras dependentes). O mesmo grupo de indivíduos é testado em duas condições diferentes.

Questão 5: Agora desempenhe o teste estatístico e reporte seus resultados. Qual seu nível de confiança e o valor estatístico crítico? Você rejeitou a hipótese nula ou falhou ao tentar rejeitá-la? Encontre uma conclusão em relação ao experimento da tarefa. Os resultados estão de acordo com suas expectativas?

Resposta: Inicialmente calculamos as médias das amostras, $x_c$ e $x_{inc}$, média para amostra congruente e média para amostra incongruente, respectivamente. Calculamos também a diferença da média: $\mu_c$ - $\mu_{inc}$. Em seguinda calculamos o desvio-padrão para cada amostra: $\sigma_c$ e $\sigma_{inc}$, desvio-padrão para amostra congruente e desvio-padrão para amostra incongruente, respectivamente. O próximo passo foi calcular o desvio-padrão das diferenças $SD$ e o desvio-padrão da média $SEM$. Com essas informações, obtivemos a estatística $t$. Também, consultando a tabela t encontramos o valor $t$-crítico.

Para o intervalo de confiança de $95$%, monocaudal, com $\alpha=0,05$ e grau de liberdade $23$, temos: $IC = (-16,30 ; 0,37)$.

Hipótese nula: $H_0$: O tempo que demora para o indivíduo falar os nomes em uma lista das cores de tamanhos iguais não muda significativamente nas duas condições. ($\mu_c$ = $\mu_{inc}$, onde: $\mu_c$ é a média de tempo para condição congruente e $\mu_{inc}$ a média de tempo para a condição incongruente.)

Hipótese alternativa: $H_A$: O tempo que demora para o indivíduo falar os nomes em uma lista das cores de tamanhos iguais é maior na condição incongruente. ($\mu_c$ < $\mu_{inc}$, onde: $\mu_c$ é a média de tempo para condição congruente e $\mu_{inc}$ a média de tempo para a condição incongruente.)

Com base na estatística $t$ encontrada ($-8,02$), valor $t$-crítico ($1,714$) e valor $p$ calculado ($ < .00001$), rejeitamos a hipótese nula.

Questão 6: Opcional: O que você acha que é responsável pelo efeito observado? Consegue pensar em uma alternativa ou tarefa similar que resultaria em um efeito parecido?

Resposta: Estudos indicam que é um fenômeno neurológico relacionado a chamada atenção direta. O efeito Stroop enfatiza a interferência que o processamento automático de palavras tem sobre um maior esforço em dizer apenas o nome da cor, isto é, a tarefa de dar uma resposta adequada quando se tem sinais conflitantes no cérebro (semântica da cor diferente do nome apresentado).

Uma tarefa similar que pode ser testada é utilizar palavras na ordem inversa da escrita ou rotacionadas 180 graus (de cabeça para baixo). Por exemplo: <font color=blue>azul</font> seria apresentado como <font color=blue>luza</font> ou mesmo outra cor, como <font color=red>luza</font>. Além disso, seguir o método, agora com essas opções acrescentadas: palavras neutras, congruentes, incongruentes e invertida (conflitante).
